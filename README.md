# 🎵 SwaraLanka — Sri Lanka's #1 Free Sinhala Music Notation Portal

> The largest collection of free Sinhala music notations for piano, guitar, violin, flute and more.

![SwaraLanka](https://img.shields.io/badge/Status-Live-brightgreen) ![License](https://img.shields.io/badge/License-MIT-blue) ![Free](https://img.shields.io/badge/Cost-100%25%20Free-gold)

## 🌐 Live Demo
👉 **[View Live on GitHub Pages](https://dananjayanadun.github.io/swaralanka/)**

## 📸 Features

- 🎹 **5,000+ Sinhala song notations** — Piano, Guitar, Violin, Flute & more
- 🔍 **Advanced search & filters** — by song, artist, instrument, difficulty, genre
- 🎓 **3 difficulty levels** — Easy, Medium, Hard — clearly marked
- 🎼 **Western & Solfège notation** — toggle between C D E and Do Re Mi
- 📱 **Mobile-first design** — works perfectly on any device
- 🖨 **Print & PDF download** — take notations anywhere
- 👥 **Community submissions** — any musician can contribute
- 🆓 **100% Free. Forever.** — no ads, no paywalls, no subscriptions

## 🚀 Getting Started

### Run Locally
```bash
# Clone the repo
git clone https://github.com/YOUR-USERNAME/swaralanka.git

# Open in browser
cd swaralanka
open index.html
```

### Deploy to GitHub Pages
1. Push to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/ (root)`
4. Your site is live at `https://dananjayanadun.github.io/swaralanka`
